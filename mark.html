Tag HTML <mark>

A tag <mark> é usada para representar texto realçado — normalmente com fundo amarelo — indicando que aquele trecho é relevante ou corresponde a um destaque no conteúdo (por exemplo, resultado de busca).

Exemplo

Exemplo simples mostrando como usar a tag <mark> para realçar partes de um parágrafo: 


<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="utf-8">
  <title>Exemplo <mark></title>
</head>
<body>

  <h1>Exemplo da tag <mark></h1>

  <p>Este é um parágrafo com uma palavra <mark>realçada</mark> para chamar a atenção do leitor.</p>

  <p>Resultados da pesquisa: <mark>HTML</mark> <mark>CSS</mark> <mark>JavaScript</mark>.</p>

</body>
</html>

Definição e uso

A tag <mark> indica texto que é relevante ou merece destaque no contexto atual. O estilo padrão dos navegadores costuma aplicar um fundo amarelo ao texto marcado, mas isso pode ser alterado via CSS.

É ideal para realçar termos em resultados de busca, instruções importantes, ou para chamar atenção a trechos específicos sem alterar a semântica do documento.

Use <mark> quando o destaque tiver significado contextual (por exemplo, "termo pesquisado encontrado aqui"), e não apenas para estilizar texto. Para destaque puramente visual, prefira classes CSS com <span>.

Boas práticas

  • Use semanticamente: destaque termos que têm relevância contextual.
  • Evite múltiplos trechos marcados sem necessidade — muitos destaques podem confundir o leitor.
  • Combine com CSS para garantir contraste e acessibilidade (verificar contraste de cor para leitores com baixa visão).
  • Para controles dinâmicos (ex.: destaque de termos de pesquisa), atualize o DOM via JavaScript adicionando/removendo <mark> ou aplicando classes.

Características e atributos

Propriedade Descrição
Tipo de elemento Elemento em linha (inline).
Uso semântico Indicar texto com importância contextual — geralmente realçado visualmente.
Estilo padrão Fundo amarelo no navegador; pode ser sobrescrito via CSS.
Atributos Não possui atributos específicos; aceita atributos globais (class, id, title, etc.).
Conteúdo permitido Texto e elementos em linha.

Acessibilidade

Certifique-se de que o destaque forneça contraste suficiente entre o texto e o fundo para ser legível por usuários com deficiência visual. Se você alterar o estilo padrão via CSS, mantenha o significado semântico do elemento.
