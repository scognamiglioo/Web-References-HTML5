Tag HTML <kbd>

A sintaxe consiste em envolver a representação de uma entrada de teclado com a tag de abertura <kbd> e a de fechamento </kbd>.

Exemplo simples

O elemento <kbd> é usado para indicar que um trecho de texto representa uma entrada do usuário, geralmente via teclado. Por padrão, os navegadores exibem seu conteúdo em uma fonte monoespaçada.

<p>Para salvar o documento, pressione <kbd>Ctrl</kbd> + <kbd>S</kbd>.</p>

Definição e uso

O elemento <kbd> (Keyboard Input) representa uma entrada do usuário. Embora seja comumente usado para teclas de teclado, ele pode semanticamente representar outras formas de entrada, como comandos de voz ou entradas de texto genéricas.

O uso da tag <kbd> ajuda a distinguir claramente as instruções de entrada do usuário do texto normal, melhorando a legibilidade e a acessibilidade de tutoriais, manuais e guias de ajuda.

  • Teclas de atalho: Indicar combinações de teclas, como <kbd>Alt</kbd> + <kbd>Tab</kbd>.
  • Navegação em menus: Descrever um caminho, como <kbd>Arquivo</kbd> → <kbd>Salvar como...</kbd>.
  • Entrada de texto: Mostrar o texto exato que o usuário deve digitar, como "Digite <kbd>exit</kbd> e pressione Enter".
  • Comandos de voz: Representar um comando de voz, como "Diga <kbd>Ok Google</kbd>".

Atributos

O elemento <kbd> não possui atributos específicos e apenas aceita os atributos globais do HTML.

Atributo Valor Descrição
id Texto Define um identificador único para o elemento.
class Texto Permite associar o elemento a uma classe CSS para estilização (ex: para parecer uma tecla real).

Boas práticas e observações

  • Diferença entre <kbd>, <code> e <samp>: É crucial usar a tag semanticamente correta. Use <kbd> para entrada do usuário, <code> para código de computador e <samp> para saída de um programa.
  • Aninhamento para combinações: Para representar uma combinação de teclas ou uma sequência, é uma boa prática aninhar as tags <kbd>. A tag externa representa a entrada completa, e as internas representam as teclas individuais.
  • Estilização: É muito comum estilizar a tag <kbd> com CSS para que ela se pareça com uma tecla física, usando bordas, sombra e um fundo claro para melhorar a experiência visual.
  • Acessibilidade: O uso correto da tag ajuda tecnologias assistivas a identificar e anunciar corretamente as instruções para o usuário.

Exemplos de uso

1) Pressionando uma única tecla

A forma mais básica, indicando uma tecla individual.

<p>Para continuar, pressione a tecla <kbd>Enter</kbd>.</p>

2) Combinação de teclas (aninhamento)

Mostra o aninhamento para representar um atalho de teclado completo.

<p>Para forçar a atualização da página, use <kbd><kbd>Ctrl</kbd>+<kbd>F5</kbd></kbd>.</p>

3) Sequência de comandos em um menu

Descrevendo passo a passo a navegação em uma interface gráfica.

<p>Vá para <kbd>Arquivo</kbd> > <kbd>Preferências</kbd> > <kbd>Configurações</kbd> para abrir o editor.</p>

4) Distinção semântica (kbd, code, samp)

Exemplo mostrando o uso correto de cada tag em um mesmo contexto.

<p>Abra seu terminal e digite o comando <kbd>ping 127.0.0.1</kbd>. O programa irá rodar o código <code>send_icmp_echo()</code> e você verá uma saída similar a esta:</p>
<pre><samp>Disparando contra 127.0.0.1 com 32 bytes de dados:
Resposta de 127.0.0.1: bytes=32 tempo<1ms TTL=128</samp></pre>
