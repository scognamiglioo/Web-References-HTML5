Tag HTML <table>

Exemplo

Cria uma tabela básica para exibir nomes e idades.

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>Maria</td>
    <td>30</td>
  </tr>
  <tr>
    <td>João</td>
    <td>25</td>
  </tr>
</table>

Definição e uso

A tag <table> define uma tabela HTML.

Uma tabela HTML consiste na tag <table> e uma ou mais tags <tr>, <th> e <td>:

  • <tr>: define uma linha da tabela (table row).
  • <th>: define uma célula de cabeçalho (table header). O texto fica em negrito e centralizado por padrão.
  • <td>: define uma célula de dados (table data).

Para maior semântica e estilização, as tabelas também podem incluir os elementos <caption>, <colgroup>, <thead>, <tfoot>, e <tbody>.

Atributos

Muitos atributos da tag <table> foram descontinuados no HTML5, e a estilização agora deve ser feita preferencialmente com CSS. No entanto, alguns ainda são vistos em código legado.

Atributo Valor Descrição
border pixels (Descontinuado em HTML5) Especifica a largura da borda ao redor da tabela.
cellpadding pixels (Descontinuado em HTML5) Define o espaçamento entre o conteúdo da célula e suas bordas.
cellspacing pixels (Descontinuado em HTML5) Define o espaçamento entre as células da tabela.
width pixels ou % (Descontinuado em HTML5) Especifica a largura da tabela.

Nota: Para controlar a aparência da tabela, use propriedades CSS como `border`, `padding`, `border-spacing` e `width`.

Outros exemplos

Tabela com cabeçalho (`thead`), corpo (`tbody`) e rodapé (`tfoot`):

<table>
  <thead>
    <tr>
      <th>Mês</th>
      <th>Vendas</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Janeiro</td>
      <td>R$ 50.000</td>
    </tr>
    <tr>
      <td>Fevereiro</td>
      <td>R$ 80.000</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>R$ 130.000</td>
    </tr>
  </tfoot>
</table>

Tabela usando `colspan` para mesclar colunas:

<table>
  <tr>
    <th colspan="2">Nome Completo</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>Silva</td>
  </tr>
</table>
