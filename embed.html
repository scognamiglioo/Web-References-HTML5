Tag HTML <embed>

Definição e uso

A tag <embed> é usada para incorporar conteúdo externo, como vídeos, áudio, PDFs, entre outros, diretamente em uma página da web.

O atributo src especifica a URL do recurso a ser incorporado. O atributo type especifica o tipo MIME do recurso.

Atenção

  • A maioria dos navegadores não suporta mais Applets Java e Plug-ins.
  • Controles ActiveX não são mais suportados em nenhum navegador.
  • O suporte ao Shockwave Flash também foi desativado nos navegadores modernos.

Boas Práticas

  • Considere usar a tag <iframe> para incorporar conteúdo externo, pois ela é mais amplamente suportada e oferece maior flexibilidade.
  • Use a tag <video> para incorporar vídeos e a tag <audio> para incorporar áudio
  • Use a tag <img> para incorporar imagens.

Atributos

Atributo Valor Descrição
height pixels Especifica a altura do conteúdo incorporado.
src URL Especifica o endereço do arquivo externo a ser incorporado.
width pixels Especifica a largura do conteúdo incorporado.

Outros exemplos

Como usar uma imagem:

<embed src="image.jpg" type="image/jpeg" width="600" height="400">

Como usar um vídeo:

<embed src="video.mp4" type="video/mp4" width="600" height="400">

Como usar um áudio:

<embed src="audio.mp3" type="audio/mpeg" width="300" height="32">

Como usar um HTML:

<embed src="page.html" type="text/html" width="600" height="400">
