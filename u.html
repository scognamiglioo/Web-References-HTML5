Tag HTML <u>

A sintaxe consiste em envolver o texto que deve ser anotado com a tag de abertura <u> e a de fechamento </u>.

Exemplo simples

O elemento <u> (Unarticulated Annotation) renderiza o texto com um sublinhado. Seu uso em HTML5 é para marcar anotações não-textuais, como a indicação de um erro de ortografia.

<p>O corretor marcou a palavra <u>paralizado</u> como incorreta.</p>

Definição e uso

O elemento <u> teve seu significado redefinido no HTML5. Originalmente, era uma tag puramente de estilo para sublinhar texto, mas essa função agora pertence ao CSS. Atualmente, seu propósito semântico é representar um trecho de texto que deve ser renderizado de forma diferente do normal, sem implicar ênfase ou importância, para indicar uma anotação não-textual.

Seu uso é bastante específico e deve ser evitado na maioria dos casos para não confundir os usuários, que associam o sublinhado a hyperlinks.

  • Erros ortográficos: É o caso de uso mais comum, onde um processador de texto marcaria uma palavra com um sublinhado ondulado.
  • Nomes próprios em Chinês: Em certos contextos, o sublinhado é usado para marcar nomes próprios em texto chinês.

Para qualquer outro tipo de sublinhado com fins de estilo ou decoração, a propriedade CSS text-decoration: underline; é a escolha correta.

Atributos

O elemento <u> não possui atributos específicos e apenas aceita os atributos globais do HTML.

Atributo Valor Descrição
id Texto Define um identificador único para o elemento.
class Texto Permite associar o elemento a uma classe CSS para estilização (ex: mudar a cor ou estilo do sublinhado).

Boas práticas e observações

  • EVITE USAR PARA ESTILO: Esta é a regra mais importante. O sublinhado é a convenção visual universal para links. Usar <u> para destacar ou estilizar texto pode frustrar o usuário, que tentará clicar em um texto não-interativo.
  • Não use para títulos ou ênfase: Para títulos, use as tags <h1>-<h6>. Para ênfase, use <em>. Para importância, use <strong>.
  • Alternativa correta é CSS: Se você precisa de um sublinhado por razões de design, aplique-o com CSS: .meu-elemento { text-decoration: underline; }.
  • Acessibilidade: O uso indevido da tag <u> prejudica a acessibilidade, pois pode confundir os usuários (incluindo aqueles que usam leitores de tela) sobre o que é um link e o que não é.

Exemplos de uso

1) Marcando um erro ortográfico (uso correto)

O exemplo canônico do uso semântico da tag <u> em HTML5.

<p>Na revisão, percebi que a palavra <u>"exceção"</u> estava escrita de forma errada.</p>

2) Uso incorreto para destacar (a ser evitado)

Este exemplo usa <u> para fins de destaque, o que está incorreto pois confunde o usuário.

<!-- Forma Incorreta -->
<h3><u>Nosso Produto Principal</u></h3>
<p>Este é um aviso <u>importante</u> para todos os clientes.</p>

3) A forma correta de sublinhar com CSS

Mostra como aplicar um sublinhado para fins de design sem usar a tag <u>.

<!-- CSS -->
.titulo-destaque {
  text-decoration: underline;
  text-decoration-color: #007bff;
  text-decoration-thickness: 2px;
}

<!-- HTML (Forma Correta) -->
<h3 class="titulo-destaque">Nosso Produto Principal</h3>
