Tag HTML <section>

A tag usada para agrupar conteúdo relacionado dentro de uma página, formando seções temáticas ou lógicas de um documento HTML.

Exemplo

No exemplo abaixo, utilizamos a tag <section> para dividir a página em partes temáticas, como "Introdução" e "Contato":

<section>
    <h2>Introdução</h2>
    <p>Bem-vindo ao nosso site! Aqui você encontrará várias informações úteis.</p>
</section>

<section>
    <h2>Contato</h2>
    <p>Entre em contato conosco pelo e-mail: contato@exemplo.com</p>
</section>

Definição e uso

A tag <section> define uma seção temática dentro de um documento HTML. Cada seção deve agrupar conteúdo relacionado, normalmente iniciando com um cabeçalho (<h1>–<h6>).

Ela é frequentemente usada dentro de páginas longas para organizar o conteúdo em partes distintas, facilitando tanto a leitura quanto a semântica do documento.

É importante usar <section> apenas quando o agrupamento tiver um significado semântico; caso contrário, prefira <div>.

Atributos

Atributo Valor Descrição
id Texto Identificador único da seção, útil para navegação e âncoras.
class Texto Define uma ou mais classes CSS para estilização.
style Declarações CSS Permite aplicar estilos diretamente no elemento (não recomendado para uso extenso).

Boas Práticas e Observações

  • Use para conteúdo relacionado: Utilize <section> para separar blocos de conteúdo com temas ou propósitos semelhantes.
  • Sempre inclua um cabeçalho: Cada seção deve conter um título claro com <h2> ou outro nível adequado de cabeçalho.
  • Evite uso excessivo: Se o agrupamento não tiver significado semântico, prefira <div>.
  • SEO e acessibilidade: Navegadores e leitores de tela entendem melhor a estrutura da página quando seções são usadas corretamente.

Outros exemplos

Veja como a tag <section> pode ser usada em diferentes contextos:

Exemplo 1: Estruturando um artigo

<article>
    <section>
        <h2>Introdução</h2>
        <p>Este artigo explica a importância das tags semânticas no HTML.</p>
    </section>

    <section>
        <h2>Conclusão</h2>
        <p>Usar tags como <section> melhora a acessibilidade e o SEO.</p>
    </section>
</article>

Exemplo 2: Seções em uma página de produto

<section id="descricao">
    <h2>Descrição do Produto</h2>
    <p>Um smartphone moderno com excelente desempenho.</p>
</section>

<section id="avaliacoes">
    <h2>Avaliações dos Usuários</h2>
    <p>Nota média: 4.8/5</p>
</section>
