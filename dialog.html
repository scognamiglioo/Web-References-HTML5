Tag HTML <dialog>

Exemplo

A tag <dialog> cria uma caixa de diálogo ou janela modal interativa.

<dialog id="meuDialogo">
  <p>Deseja confirmar esta ação?</p>
  <button id="fecharBtn">Fechar</button>
</dialog>

<button id="abrirBtn">Abrir Diálogo</button>

<script>
  const dialog = document.getElementById('meuDialogo');
  document.getElementById('abrirBtn').onclick = () => dialog.showModal();
  document.getElementById('fecharBtn').onclick = () => dialog.close();
</script>

Definição e uso

A tag <dialog> é usada para criar janelas de diálogo, caixas de alerta ou modais embutidas no HTML.

O elemento pode ser aberto com os métodos JavaScript show() e showModal(), e fechado com close().

É uma forma nativa e sem dependência de bibliotecas externas para criar pop-ups interativos.

Atributos

Atributo Valor Descrição
open booleano Define se o diálogo está visível. Quando presente, o diálogo é exibido automaticamente.

Outros exemplos

Exemplo de caixa de diálogo simples aberta automaticamente:

<dialog open>
  <p>Mensagem automática</p>
</dialog>

Exemplo com botão para fechar:

<dialog id="janela">
  <p>Olá! Clique abaixo para fechar.</p>
  <button onclick="janela.close()">Fechar</button>
</dialog>

<button onclick="janela.showModal()">Abrir</button>
