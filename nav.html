Tag HTML5 <nav>

Definição e uso

A tag <nav> define uma **área de navegação** em uma página HTML5, geralmente contendo links para seções importantes do site. É recomendada para menus principais, barras de navegação e links de rodapé que organizam a navegação do usuário.
Conteúdos dentro de <nav> devem ser links de navegação relevantes. Não use para links internos genéricos.

Estrutura Básica

A sintaxe típica é:
<nav><ul><li><a href="#">Link</a></li></ul></nav>

Exemplo 1: Menu de navegação horizontal

<nav>
    <ul>
       <li><a href="index.html">Home</a></li>
       <li><a href="sobre.html">Sobre</a></li>
       <li><a href="contato.html">Contato</a></li>
    </ul>
</nav>

Exemplo 2: Navegação vertical

<nav style="width: 200px;">
    <ul style="list-style: none; padding: 0;">
       <li><a href="pagina1.html">Página 1</a></li>
       <li><a href="pagina2.html">Página 2</a></li>
       <li><a href="pagina3.html">Página 3</a></li>
    </ul>
</nav>
