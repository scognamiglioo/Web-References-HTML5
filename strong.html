Tag HTML <strong>

A sintaxe requer que o texto a ser destacado seja envolvido pela tag de abertura <strong> e a de fechamento </strong>.

Exemplo simples

O elemento <strong> é usado para indicar que um texto tem grande importância, seriedade ou urgência. Os navegadores, por padrão, renderizam o conteúdo desta tag em negrito. 

<p><strong>Atenção:</strong> Este produto contém amendoim.</p>

Definição e uso

O elemento <strong> define um texto com "forte importância". A tag não é meramente estilística; ela carrega um peso semântico que informa aos navegadores e tecnologias assistivas que aquele conteúdo é mais importante que o restante.

Leitores de tela podem usar uma entonação de voz diferente para o conteúdo dentro de <strong>, ajudando a transmitir a urgência ou importância da informação para o usuário.

  • Avisos e alertas: Para destacar informações críticas que não podem ser ignoradas.
  • Urgência: Indicar prazos ou informações sensíveis ao tempo.
  • Pontos-chave: Enfatizar a parte mais importante de uma frase ou parágrafo.
  • Rótulos: Destacar o conceito principal que o texto seguinte irá descrever.

Para texto que deve ser apenas visualmente em negrito sem adicionar importância semântica (como o nome de um produto em uma análise), a tag <b> ou a propriedade CSS font-weight: bold; são mais adequadas.

Atributos

O elemento <strong> não possui atributos específicos e apenas aceita os atributos globais do HTML.

Atributo Valor Descrição
id Texto Define um identificador único para o elemento.
class Texto Permite associar o elemento a uma classe CSS para estilização.

Boas práticas e observações

  • Importância vs. Ênfase (<strong> vs. <em>): Use <strong> para indicar importância. Use <em> para dar ênfase, o que altera o significado de uma frase através da entonação.
  • Semântica vs. Estilo (<strong> vs. <b>): A tag <strong> é semântica. A tag <b> é para chamar a atenção visualmente sem adicionar importância (ex: palavras-chave). Para negrito puramente decorativo, use CSS.
  • Não use para títulos: Para criar títulos e subtítulos, use as tags de cabeçalho (<h1> a <h6>). Elas estruturam a página, o que é algo que <strong> não faz.
  • Não abuse: Usar <strong> em excesso dilui o seu propósito. Reserve a tag para as partes do texto que são genuinamente mais importantes.

Exemplos de uso

1) Alerta de segurança

Um uso clássico para indicar uma informação crítica que o usuário precisa saber.

<p><strong>Cuidado:</strong> Não compartilhe sua senha com ninguém.</p>

2) Indicando urgência

Destacando uma informação sensível ao tempo para criar um senso de urgência.

<p>Sua oferta expira em <strong>24 horas</strong>.</p>

3) Aninhamento para aumentar a importância

Aninhar uma tag <strong> dentro de outra indica um nível ainda maior de importância.

<p><strong>Aviso final: A sua conta será permanentemente excluída se nenhuma ação for tomada <strong>hoje</strong>.</strong></p>

4) Comparação semântica (strong, em, b)

Exemplo mostrando o uso correto de cada tag de acordo com seu significado.

<p><strong>Regra 1:</strong> O tópico principal do trabalho deve ser definido na primeira semana. Isso <em>não</em> é negociável. O formato de citação a ser utilizado é o <b>ABNT</b>.</p>
