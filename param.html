Tag HTML <param>

Definição e uso

A tag <param> é usada para definir parâmetros para plugins em elementos <object>.

Atributos

Atributo Valor Descrição
name name Especifica o nome do parâmetro.
value value Especifica o valor do parâmetro.

Exemplo:

Setar parametro autoplay para true, então o som vai começar tocar assim que a página carregar:

<object data="som.mp3" type="audio/mpeg">
    <param name="autoplay" value="true">
    <param name="volume" value="0.5">
</object>
