Tag HTML <a>

Exemplo

Cria um link para o site do IFNMG.

<a href="http://www.ifnmg.edu.br">Visite o site do IFNMG!</a>

Definição e uso

A tag <a> define um hyperlink, o qual é usado para ligar uma página a outra.

O atributo href especifica o destino do link. O texto entre as tags de abertura e fechamento é o texto do link que os usuários clicam.

Atributos

Atributo Valor Descrição
download filename Especifica que o recurso vinculado será baixado quando o usuário clicar no link.
href URL Especifica a URL da página ou recurso que o link aponta.
hreflang language_code Especifica o idioma do documento vinculado.
media media_query Especifica para qual mídia ou dispositivo o documento vinculado está otimizado.
ping list_of_URLs Lista de URLs que receberão uma requisição POST em segundo plano quando o link for seguido, geralmente usado para rastreamento.
referrerpolicy no-referrer / no-referrer-when-downgrade / origin / origin-when-cross-origin / same-origin / strict-origin-when-cross-origin / unsafe-url Especifica quais informações de referência serão enviadas com o link.
rel alternate / author / bookmark / external / help / license / next / nofollow / noreferrer / noopener / prev / search / tag Especifica o relacionamento entre o documento atual e o documento vinculado.
target _blank / _parent / _self / _top Especifica onde abrir o documento vinculado.
type media_type Especifica o tipo de mídia do documento vinculado.

Outros exemplos

Como usar uma imagem como link:

<a href="http://www.ifnmg.edu.br">
    <img src="ifnmg_logo.png" alt="Logo IFNMG">
</a>

Como abrir um link em uma nova aba:

<a href="https://www.w3schools.com" target="_blank">Visit W3Schools.com!</a>

Como criar um link de email:

<a href="mailto:exemplo@ifnmg.edu.br">Enviar Email</a>

Como criar um link para um número de telefone:

<a href="tel:+1234567890">Ligar para +1234567890</a>

Como criar um link para uma seção específica da mesma página:

<a href="#secao1">Ir para Seção 1</a>

<h2 id="secao1">Seção 1< ;/h2>

Como fazer link para JavaScript:

<a href="javascript:void(0);" onclick="alert('Olá! Você clicou no link.');">Clique aqui</a>
