Tag HTML <object>

A tag usada para incorporar conteúdo externo, como imagens, vídeos, PDFs e outros recursos dentro de uma página HTML.

Exemplo

No exemplo abaixo, um arquivo PDF é incorporado dentro da página usando a tag <object>: 

<object data="documento.pdf" type="application/pdf" width="600" height="400">
    Seu navegador não suporta a exibição de PDFs.
</object>

Definição e uso

A tag <object> é usada para incorporar recursos externos — como arquivos multimídia, documentos ou aplicativos — diretamente em uma página HTML.

Ela pode ser usada como alternativa para <iframe> e <embed>, mas é mais versátil por aceitar conteúdo alternativo entre suas tags.

Quando o recurso não pode ser exibido (por exemplo, se o navegador não suporta o tipo do arquivo), o conteúdo dentro do <object> é mostrado como fallback.

Atributos

Atributo Valor Descrição
data URL (Obrigatório) Especifica o caminho do recurso externo que será exibido.
type MIME type Indica o tipo de conteúdo a ser incorporado (por exemplo, image/png, application/pdf).
width / height Valor em px ou % Define as dimensões da área de exibição do objeto incorporado.
name Texto Define um nome para o objeto, que pode ser usado por scripts.
form ID do formulário Associa o objeto a um formulário HTML específico.

Boas Práticas e Observações

  • Fallback importante: Sempre forneça um texto ou conteúdo alternativo entre as tags <object> para casos em que o navegador não consiga renderizar o recurso.
  • Versatilidade: Use <object> para diversos tipos de conteúdo, como PDFs, vídeos, imagens SVG ou até aplicativos interativos.
  • Alternativa ao <embed>: A tag <object> é mais padronizada e semântica que <embed>.
  • Evite uso excessivo: Prefira <img> para imagens simples e <iframe> para páginas HTML inteiras.

Outros exemplos

Veja alguns usos práticos da tag <object>:

Exemplo 1: Incorporando um arquivo PDF

<object data="utils/doc/homepage.pdf" type="application/pdf" width="300" height="50">
Seu navegador não suporta PDF incorporado.
</object>

Exemplo 2: Exibindo um SVG

<object data="utils/imgs/cat.svg" type="image/svg+xml" width="200" height="200"></object>
