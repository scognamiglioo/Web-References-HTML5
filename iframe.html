Tag HTML <iframe>

Exemplo

A tag <iframe> é usada para incorporar outra página HTML dentro da atual.

<iframe src="https://www.example.com" width="400" height="300"></iframe>

O exemplo acima exibe o conteúdo do site indicado dentro de uma moldura.

Definição e uso

A tag <iframe> (inline frame) cria uma área retangular que carrega outro documento HTML.

É muito usada para incorporar vídeos, mapas, formulários ou outros sites dentro da página.

Também pode ser usada para incluir conteúdo dinâmico, como players do YouTube ou embeds de redes sociais.

Atributos

Atributo Valor Descrição
src URL Define a página ou recurso a ser exibido dentro do iframe.
width número Define a largura do iframe.
height número Define a altura do iframe.
name texto Permite nomear o iframe para ser alvo de links e formulários.
allowfullscreen booleano Permite que o conteúdo seja exibido em tela cheia (útil em vídeos).
loading lazy / eager Controla o carregamento do iframe (lazy = carrega quando visível).

Outros exemplos

Exemplo com vídeo do YouTube:

<iframe width="560" height="315"
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen></iframe>

Exemplo com Google Maps:

<iframe
  src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3..."
  width="400" height="300" style="border:0;" allowfullscreen="" loading="lazy"></iframe>
