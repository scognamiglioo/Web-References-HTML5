Tag HTML <ol>

Exemplo

A tag <ol> é usada para criar listas ordenadas — ou seja, numeradas.

<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>

Resultado:

  1. HTML
  2. CSS
  3. JavaScript

Definição e uso

A tag <ol> (ordered list) cria uma lista numerada, onde a ordem dos itens é importante.

Cada item da lista é definido com a tag <li> (list item).

Você pode personalizar o tipo de numeração (números, letras, algarismos romanos, etc.).

Atributos

Atributo Valor Descrição
type 1 | A | a | I | i Define o tipo de marcador da lista (número, letra ou algarismo romano).
start número Define o número inicial da contagem.
reversed booleano Inverte a ordem da contagem (de cima para baixo).

Outros exemplos

Lista começando do número 5:

<ol start="5">
  <li>Item A</li>
  <li>Item B</li>
</ol>

Lista invertida (reversed):

<ol reversed>
  <li>Primeiro</li>
  <li>Segundo</li>
</ol>

Lista com letras maiúsculas:

<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
