Tag HTML <tbody>

A tag <tbody> representa o corpo principal de uma tabela (<table>), agrupando as linhas de dados (<tr>). É usada para separar semanticamente o conteúdo tabular do cabeçalho (<thead>) e do rodapé (<tfoot>).

Exemplo

Exemplo de tabela com <thead>, <tbody> e <tfoot>, mostrando claramente onde fica o corpo da tabela:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="utf-8">
  <title>Exemplo <tbody></title>
  <style>
    table { border-collapse: collapse; width: 100%; }
    th, td { border: 1px solid #333; padding: 8px; text-align: left; }
    thead { background: #f0f0f0; }
    tfoot { background: #fafafa; font-style: italic; }
  </style>
</head>
<body>

  <h1>Tabela com <tbody></h1>

  <table>
    <thead>
      <tr>
        <th>Produto</th>
        <th>Preço</th>
        <th>Quantidade</th>
      </tr>
    </thead>

    <tbody>
      <tr>
        <td>Caneta</td>
        <td>R$ 2,50</td>
        <td>100</td>
      </tr>
      <tr>
        <td>Caderno</td>
        <td>R$ 15,00</td>
        <td>50</td>
      </tr>
    </tbody>

    <tfoot>
      <tr>
        <td colspan="3">Total de itens: 150</td>
      </tr>
    </tfoot>
  </table>

</body>
</html>

Definição e uso

A tag <tbody> agrupa as linhas de dados de uma tabela. Embora o navegador funcione sem ela (linhas <tr> podem existir diretamente dentro de <table>), usar <thead>, <tbody> e <tfoot> melhora a semântica, facilita a estilização e a manipulação via JavaScript (por exemplo, ao paginar ou ordenar linhas).

Pode haver múltiplos elementos <tbody> em uma tabela — isso permite agrupar logicamente diferentes blocos de linhas (por exemplo, por período, categoria, etc.).

Boas práticas

  • Use <thead>, <tbody> e <tfoot> para separar cabeçalho, corpo e rodapé da tabela.
  • Ao gerar tabelas dinamicamente, envolva as linhas em um único <tbody> para facilitar a substituição/atualização via DOM.
  • Considere usar múltiplos <tbody> quando quiser agrupar logicamente blocos distintos de linhas.
  • Combine com aria-describedby ou outras técnicas de acessibilidade para tabelas complexas.

Características e atributos

Propriedade Descrição
Local permitido Dentro de <table>. Deve estar entre <thead> e <tfoot> se eles existirem.
Conteúdo permitido Um ou mais elementos <tr> (linhas da tabela).
Múltiplos elementos Permitido — pode usar mais de um <tbody> para agrupar blocos de linhas.
Atributos Não possui atributos específicos além dos globais (como id, class, style).
Uso com CSS/JS Útil para aplicar estilos a blocos de linhas ou para manipulação via script (ex.: paginação, ordenação).

Acessibilidade

Para tabelas complexas, forneça cabeçalhos claros (<th>) e associe-os a células via atributos como scope ou headers. Separar o corpo com <tbody> ajuda leitores de tela e scripts a localizar rapidamente o conteúdo de dados.
