Tag HTML <em>

Você utiliza uma tag de abertura <em> e uma de fechamento </em> para envolver o texto que receberá ênfase.

Exemplo simples

O elemento <em> é usado para aplicar ênfase a uma parte do texto. Por padrão, os navegadores renderizam o conteúdo dentro da tag <em> em itálico.

<p>Gatos são <em>geralmente</em> mais independentes que cachorros.</p>

Definição e uso

O elemento <em> (Emphasis) marca um texto que possui uma ênfase especial em relação ao conteúdo ao redor. O significado da frase pode mudar dependendo da palavra enfatizada.

Essa tag não é apenas para estilização; ela tem um propósito semântico. Leitores de tela podem alterar a entonação da voz ao ler o texto contido em <em>, transmitindo a ênfase para usuários com deficiência visual.

  • Alterar o sentido: Usado para indicar qual palavra de uma frase é o foco principal.
  • Contraste: Para contrastar um termo com outro.
  • Definições ou termos técnicos: Frequentemente usado ao introduzir um conceito pela primeira vez.

Para fins puramente visuais de itálico, sem carga semântica, a tag <i> ou a propriedade CSS font-style: italic; são mais apropriadas.

Atributos

O elemento <em> não possui atributos específicos e apenas aceita os atributos globais do HTML.

Atributo Valor Descrição
id Texto Define um identificador único para o elemento.
class Texto Permite associar o elemento a uma classe CSS para estilização.

Boas práticas e observações

  • Ênfase vs. Importância: Use <em> para dar ênfase (mudança de entonação). Para indicar que algo é importante, sério ou urgente (conteúdo de grande importância), use a tag <strong>.
  • Semântica vs. Estilo: Não use <em> apenas para obter o efeito de itálico. Se o texto não precisa de ênfase semântica, use a propriedade CSS font-style: italic;.
  • Aninhamento: É possível aninhar tags <em> para indicar um nível maior de ênfase. Cada nível de aninhamento aumenta a importância da ênfase.
  • Alternativa ao <i>: Enquanto <em> indica ênfase, a tag <i> é usada para texto que se diferencia do normal, como um termo técnico, um nome de navio ou uma palavra em outro idioma, sem implicar ênfase ou importância.

Exemplos de uso

1) Alterando o foco de uma frase

Note como a ênfase em palavras diferentes muda o que está sendo comunicado.

<p>Eu <em>não</em> disse que ele roubou o dinheiro.</p> <!-- Outra pessoa disse. -->
<p>Eu não disse que <em>ele</em> roubou o dinheiro.</p> <!-- Outra pessoa roubou. -->
<p>Eu não disse que ele roubou o <em>dinheiro</em>.</p> <!-- Ele roubou outra coisa. -->

2) Aninhando para maior ênfase

O aninhamento aumenta o nível de ênfase na palavra "única".

<p>Esta é a <em>única</em> forma de resolver o problema.</p>
<p>Não, esta é <em>a <em>única</em></em> forma de resolver o problema!</p>

3) Comparação com <strong> e <i>

Exemplo prático mostrando a diferença semântica entre as tags.

<p>A <i>Felis catus</i> é uma espécie popular de animal de estimação.</p> <!-- Termo técnico, sem ênfase -->
<p>Você precisa alimentar o gato <em>todos os dias</em>.</p> <!-- Ênfase na frequência -->
<p><strong>Atenção:</strong> Não dê chocolate para seu gato.</p> <!-- Alerta importante -->
