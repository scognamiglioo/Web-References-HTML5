Tag HTML <p>

A sintaxe é simples: você utiliza uma tag de abertura <p> e uma tag de fechamento </p>. O texto que compõe o parágrafo é inserido entre elas.

Exemplo simples

Neste exemplo, mostramos como parágrafos são usados para estruturar blocos de texto em uma página:

<p>Este é o primeiro parágrafo. Ele contém texto sobre um determinado assunto, separado logicamente de outros conteúdos.</p>
<p>Este é o segundo parágrafo. Usar múltiplos parágrafos melhora a legibilidade e a organização do conteúdo.</p>

Output

Este é o primeiro parágrafo. Ele contém texto sobre um determinado assunto, separado logicamente de outros conteúdos.

Este é o segundo parágrafo. Usar múltiplos parágrafos melhora a legibilidade e a organização do conteúdo.

Definição e uso

O elemento <p> é usado para definir um parágrafo de texto. Navegadores automaticamente adicionam um espaço (margem) antes e depois de cada elemento <p>, separando-o visualmente de outros blocos.

Semanticamente, <p> agrupa um bloco de texto que forma uma unidade lógica de pensamento. Isso é fundamental para:

  • Acessibilidade: Leitores de tela usam parágrafos para pausar a leitura e permitir que o usuário navegue pelo texto de forma estruturada.
  • Estilização com CSS: Facilita a aplicação de estilos como espaçamento, indentação da primeira linha, alinhamento e altura da linha a blocos de texto específicos.
  • Manutenção do código: Um texto bem estruturado em parágrafos é mais fácil de ler e editar no código-fonte.

Atributos

O elemento <p> suporta principalmente os atributos globais do HTML, que são comuns a quase todas as tags.

Atributo Valor Descrição
id Texto Define um identificador único para o parágrafo, útil para links âncora e manipulação com JavaScript.
class Texto Associa o parágrafo a uma ou mais classes CSS para estilização.
style Regras CSS Aplica estilos CSS diretamente no elemento (inline), embora o uso de classes seja geralmente preferível.
title Texto Fornece informações extras sobre o parágrafo, que geralmente aparecem como uma dica (tooltip) ao passar o mouse sobre ele.

Observação: O atributo align (ex: <p align="center">) foi depreciado em HTML5. Para alinhar texto, use a propriedade CSS text-align.

Boas práticas e observações

  • Não use para espaçamento: Evite usar parágrafos vazios (<p></p>) apenas para criar espaço vertical. Use as propriedades CSS margin ou padding para isso.
  • Uso semântico: Use <p> para blocos de texto que realmente formam um parágrafo. Para listas, use <ul> ou <ol>; para citações, use <blockquote>.
  • Fechamento da tag: Embora alguns navegadores possam renderizar corretamente sem a tag de fechamento </p>, a especificação HTML exige que ela seja fechada. Sempre feche suas tags.
  • Elementos aninhados: Um parágrafo pode conter elementos "inline" como <strong>, <em>, <a> e <span>, mas não pode conter elementos de bloco, como outros parágrafos, cabeçalhos (<h1>-<h6>) ou listas.

Exemplos de uso

1) Parágrafo com elementos inline

Aqui, usamos <strong> para dar ênfase e <a> para criar um link dentro do parágrafo.

<p>Para mais informações, <strong>leia nossa documentação</strong> ou <a href="/contato">entre em contato</a> conosco.</p>

2) Estilização de parágrafo com CSS

Este exemplo mostra como aplicar um estilo específico a um parágrafo usando uma classe.

<!-- CSS -->
.paragrafo-destaque {
  background-color: #f0f0f0;
  border-left: 5px solid #007bff;
  padding: 15px;
  font-style: italic;
}

<!-- HTML -->
<p class="paragrafo-destaque">Esta é uma nota importante que precisa de atenção especial do leitor.</p>

Output

Esta é uma nota importante que precisa de atenção especial do leitor.
