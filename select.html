Tag HTML <select>

Exemplo

A tag <select> cria uma lista suspensa (dropdown) de opções.

<label for="linguagem">Escolha uma linguagem:</label>
<select id="linguagem" name="linguagem">
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
</select>

Definição e uso

A tag <select> é usada para criar menus suspensos, permitindo que o usuário escolha uma ou mais opções.

Cada item dentro do <select> é definido por uma tag <option>.

É comum ser usada em formulários, junto com a tag <label> para identificar o campo.

Atributos

Atributo Valor Descrição
name texto Define o nome do campo, usado no envio de formulários.
multiple booleano Permite selecionar múltiplas opções (usando Ctrl ou Shift).
size número Define quantas opções são visíveis sem rolar.
disabled booleano Desativa o campo, impedindo interação.

Outros exemplos

Menu com múltiplas seleções possíveis:

<select multiple size="3">
  <option>PHP</option>
  <option>Python</option>
  <option>Java</option>
</select>

Grupo de opções categorizadas:

<select>
  <optgroup label="Front-end">
    <option>HTML</option>
    <option>CSS</option>
  </optgroup>
  <optgroup label="Back-end">
    <option>Node.js</option>
    <option>Python</option>
  </optgroup>
</select>
