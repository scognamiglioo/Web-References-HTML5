Tag HTML <area>

A tag usada para definir áreas clicáveis dentro de uma imagem mapeada no HTML.

Exemplo

No exemplo abaixo, criamos uma imagem com áreas clicáveis definidas por coordenadas dentro de um mapa de imagem:

                            <img src="planeta.jpg" alt="Mapa dos planetas" usemap="#planetmap">

                            <map name="planetmap">             
                                <area shape="circle" coords="180,139,60" href="mercurio.html" alt="Mercúrio">
                                <area target="_blank" alt="sol" title="sol" href="https://en.wikipedia.org/wiki/Sun" coords="246,246,73" shape="circle"> 
                                <area target="_blank" alt="saturno" title="saturno" href="https://en.wikipedia.org/wiki/Saturn" coords="97,313,33" shape="circle"> 
                                <area target="_blank" alt="urano" title="urano" href="https://en.wikipedia.org/wiki/Uranus" coords="195,394,28" shape="circle"> 
                                <area target="_blank" alt="netuno" title="netuno" href="https://en.wikipedia.org/wiki/Neptune" coords="312,391,31" shape="circle"> 
                                <area target="_blank" alt="jupiter" title="jupiter" href="https://en.wikipedia.org/wiki/Jupiter" coords="406,287,50" shape="circle"> 
                                <area target="_blank" alt="terra" title="terra" href="https://en.wikipedia.org/wiki/Earth" coords="263,93,25" shape="circle"> 
                                <area target="_blank" alt="venus" title="venus" href="https://en.wikipedia.org/wiki/Venus" coords="143,118,19" shape="circle"> 
                                <area target="_blank" alt="marte" title="marte" href="https://en.wikipedia.org/wiki/Mars" coords="386,164,21" shape="circle"> 
                                <area target="_blank" alt="mercurio" title="mercurio" href="https://en.wikipedia.org/wiki/Mercury_(planet)" coords="97,213,14" shape="circle"> 
                            </map>

Neste exemplo, cada <area> representa uma parte clicável diferente da imagem.

Output

Experimente clicar em um dos planetas da imagem.

Definição e uso

A tag <area> define uma região interativa dentro de um mapa de imagem (<map>). Essas áreas permitem criar links em partes específicas de uma imagem, como em um diagrama, mapa geográfico ou infográfico.

Cada <area> é definida por coordenadas que especificam sua posição e formato dentro da imagem. Ela deve estar sempre dentro de um elemento <map> e associada a uma imagem por meio do atributo usemap.

Atributos

Atributo Valor Descrição
alt Texto alternativo Descreve a área; importante para acessibilidade.
coords Lista de números Define as coordenadas que determinam a posição e o formato da área dentro da imagem.
download Nome do arquivo Indica que o link deve fazer download do destino em vez de abri-lo.
href URL Define o destino do link ao clicar na área.
hreflang Código de idioma Indica o idioma do documento de destino.
media Media query Define para quais dispositivos ou condições de mídia o link é otimizado.
referrerpolicy no-referrer, origin, same-origin, etc. Controla quais informações de referência (referrer) são enviadas ao acessar o link.
rel Tipo de relação Define a relação entre o documento atual e o destino do link.
shape rect, circle, poly, default Define o formato da área clicável.
target _blank, _self, etc. Especifica onde abrir o link.
type Tipo MIME. Define o tipo de mídia (MIME type) do recurso de destino.

Boas Práticas e Observações

  • Uso Correto: Sempre use <area> dentro de um <map>.
  • Acessibilidade: Forneça sempre o atributo <alt> para descrever a área.
  • Coordenadas: Ajuste as coordenadas de acordo com o tamanho real da imagem. Se a imagem for redimensionada via CSS, o mapeamento pode perder a precisão.
  • Forma adequada: Use shape="rect" para retângulos, shape="circle" para círculos e shape="poly" para formas irregulares.
  • SEO e usabilidade: Evite usar apenas imagens mapeadas como meio principal de navegação; ofereça alternativas textuais.

Outros exemplos

Veja alguns usos práticos da tag <area>:

Exemplo 1: Área retangular

<area shape="rect" coords="34,44,270,350" href="produto.html" alt="Produto">

Output

Cria uma área clicável retangular na imagem.

Exemplo 2: Área circular

<area shape="circle" coords="180,139,60" href="informacoes.html" alt="Informações">

Output

Cria uma área clicável circular com raio de 60 pixels.

Exemplo 3: Área poligonal

<area shape="poly" coords="140,121,181,116,204,164,121,198" href="setor.html" alt="Setor do mapa">

Output

Cria uma área clicável com formato personalizado de acordo com os pontos definidos.
