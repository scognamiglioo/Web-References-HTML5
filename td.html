Tag HTML <td>

A tag usada para definir células de dados dentro de uma linha de uma tabela HTML.

Exemplo

No exemplo abaixo, a tag <td> é usada para preencher as células de uma tabela simples:

                            <table border="1">
                            <tr>
                                <td>Nome</td>
                                <td>Idade</td>
                            </tr>
                            <tr>
                                <td>Ana</td>
                                <td>22</td>
                            </tr>
                            </table>

Neste exemplo, cada <td> representa uma célula comum da tabela. O primeiro <tr> contém os títulos das colunas e o segundo contém os dados.

Output

Nome Idade
Ana 22

Definição e uso

A tag <td> (table data) é usada para definir uma célula de dados em uma tabela. Ela deve estar sempre dentro de uma tag <tr> (linha da tabela) e, normalmente, dentro de uma <table>.

Os conteúdos de <td> podem ser texto, imagens, links, listas, formulários ou qualquer outro elemento HTML.

Atributos

Atributo Valor Descrição
colspan número Define quantas colunas a célula deve ocupar.
rowspan número Define quantas linhas a célula deve ocupar.
headers id(s) de <th> Associa a célula a cabeçalhos específicos (para acessibilidade).

Boas Práticas e Observações

  • Estrutura: Use <td> para dados comuns e <th> para cabeçalhos de tabela.
  • Semântica: Prefira CSS para definir estilos e alinhamentos em vez de atributos obsoletos (align, valign, bgcolor).
  • Acessibilidade: Quando a tabela for complexa, use os atributos headers e scope para melhorar a leitura por leitores de tela.
  • Estilização: O CSS pode controlar bordas, cores, espaçamento e alinhamento com propriedades como border, padding, text-align e vertical-align.
  • Boa prática: Sempre feche a tag <td> explicitamente, mesmo que alguns navegadores a fechem automaticamente.

Outros exemplos

Veja alguns usos práticos da tag <td>:

Exemplo 1: Mesclando colunas

                            <table border="1">
                                <tr>
                                    <td colspan="2">Informações Pessoais</td>
                                </tr>
                                <tr>
                                    <td>Nome</td>
                                    <td>Maria</td>
                                </tr>
                            </table>

Output

Informações Pessoais
Nome Maria

A primeira célula ocupa duas colunas.

Exemplo 2: Mesclando linhas

                            <table border="1">
                                <tr>
                                    <td rowspan="2">Endereço</td>
                                    <td>Rua A</td>
                                </tr>
                                <tr>
                                    <td>Rua B</td>
                                </tr>
                            </table>

Output

Endereço Rua A
Rua B

A célula “Endereço” ocupa duas linhas

Exemplo 3: Célula com conteúdo misto

                            <table border="1">
                                <tr>
                                    <td>
                                    <strong>Foto:</strong> Sistema solar<br>
                                    <img src=""utils/imgs/planeta.jpg" width="60" alt="solar">
                                    </td>
                                </tr>
                            </table>

Output

Foto: Sistema Solar

A célula contém texto e imagem.
