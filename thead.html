Tag HTML <thead>

Elemento usado para agrupar as linhas do cabeçalho de uma tabela HTML. Ajuda na organização semântica, acessibilidade e estilização de tabelas complexas.

Exemplo simples

Neste exemplo, o cabeçalho da tabela está dentro de <thead> e o conteúdo dentro de <tbody>:

<table>
  <thead>
    <tr>
      <th>Produto</th>
      <th>Preço</th>
      <th>Disponível</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Fone de ouvido</td>
      <td>R$ 79,90</td>
      <td>Sim</td>
    </tr>
    <tr>
      <td>Teclado</td>
      <td>R$ 149,00</td>
      <td>Não</td>
    </tr>
  </tbody>
</table>

Output

Exemplo: Produtos
Produto Preço Disponível
Fone de ouvido R$ 79,90 Sim
Teclado R$ 149,00 Não

Definição e uso

O elemento <thead> é usado para agrupar as linhas de cabeçalho de uma tabela. Ele deve conter uma ou mais linhas (<tr>), e essas linhas costumam incluir células de cabeçalho (<th>).

Semânticamente, <thead> serve para separar a parte de cabeçalho do corpo da tabela (<tbody>) e do rodapé (<tfoot>), o que facilita:

  • a leitura por tecnologias assistivas (leitores de tela);
  • a estilização via CSS (por exemplo: aplicar estilos apenas ao cabeçalho);
  • a repetição automática de cabeçalhos em situações de impressão ou em tabelas com scroll (dependendo da implementação do navegador/estilos).

Atributos

O elemento <thead> não possui atributos específicos além dos atributos globais (como id, class, style, etc.). Em termos de acessibilidade, é comum usar atributos nos elementos <th>, como scope e headers.

Atributo Valor Descrição
id Texto Identificador único da seção de cabeçalho (útil para scripts ou âncoras).
class Texto Classes CSS para estilização do cabeçalho.
role Texto Normalmente rowgroup — pode ser usado para explicitar o papel para tecnologias assistivas.

Observação: atributos relevantes para acessibilidade como scope="col" ou scope="row" devem ser aplicados às células <th>, não ao <thead> diretamente.

Boas práticas e observações

  • Use <thead> sempre que houver cabeçalho lógico: facilita a leitura e a manutenção do HTML.
  • Coloque títulos de coluna dentro de <th> e use scope: <th scope="col">Nome</th> ajuda leitores de tela a mapear cabeçalhos para células.
  • Para cabeçalhos compostos (agrupados): use múltiplas linhas dentro de <thead> (ver exemplo abaixo).
  • Evite misturar conteúdo estrutural e visual: não use <thead> apenas para estilizar; ele tem significado semântico.
  • Impressão e scroll: alguns navegadores podem repetir o conteúdo de <thead> em páginas impressas; para headers fixos (sticky) prefira soluções via CSS com cuidado.

Exemplos avançados

1) Cabeçalho com múltiplas linhas (grupos de colunas)

<table>
  <thead>
    <tr>
      <th rowspan="2">Produto</th>
      <th colspan="2">Vendas (Q1)</th>
      <th rowspan="2">Total</th>
    </tr>
    <tr>
      <th>Janeiro</th>
      <th>Fevereiro</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Mouse</td>
      <td>120</td>
      <td>95</td>
      <td>215</td>
    </tr>
  </tbody>
</table>

2) Cabeçalho com acessibilidade (usando scope)

<table>
  <thead>
    <tr>
      <th scope="col">Nome</th>
      <th scope="col">Email</th>
      <th scope="col">Telefone</th>
    </tr>
  </thead>
  <tbody> ... </tbody>
</table>

3) Cabeçalho fixo (sticky) — exemplo de CSS

Este exemplo mostra a técnica mais comum para deixar o cabeçalho fixo enquanto o corpo da tabela rola. Atenção: para funcionar corretamente, o contêiner pai pode precisar de overflow e a tabela deve ter largura definida. 

<!-- CSS -->
.table-container {
  max-height: 300px;
  overflow: auto;
}

.table-container thead th {
  position: sticky;
  top: 0;
  background: #fff; /* garanta contraste */
  z-index: 2;
}

<!-- HTML -->
<div class="table-container">
  <table>
    <thead> ... </thead>
    <tbody> ... </tbody>
  </table>
</div>

Nota: o comportamento exato pode variar entre navegadores e depende de estilos como border-collapse e do fluxo de layout. Teste em diferentes navegadores e dispositivos.

Exemplo completo: tabela com <thead>, <tbody> e <tfoot>

<table>
  <caption>Relatório de Vendas</caption>
  <thead>
    <tr>
      <th scope="col">Mês</th>
      <th scope="col">Vendas</th>
      <th scope="col">Crescimento</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Janeiro</td><td>R$ 10.000</td><td>+5%</td></tr>
    <tr><td>Fevereiro</td><td>R$ 12.000</td><td>+20%</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>R$ 22.000</td><td> </td></tr>
  </tfoot>
</table>

Output

Relatório de Vendas
Mês Vendas Crescimento
Janeiro R$ 10.000 +5%
Fevereiro R$ 12.000 +20%
Total R$ 22.000
