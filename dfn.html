Tag HTML <dfn>

A tag usada para indicar o termo que está sendo definido em um texto.

Exemplo

No exemplo abaixo, o termo “HTML” é destacado como o termo sendo definido, usando a tag <dfn>: 

<p>
    <dfn>HTML</dfn> (HyperText Markup Language) é a linguagem padrão para criar páginas da web.
</p>

Definição e uso

A tag <dfn> identifica um termo que está sendo definido no contexto do documento. É útil para leitores, motores de busca e tecnologias assistivas entenderem que o texto marcado é o nome de um conceito que está sendo explicado.

Quando usada junto com o elemento <abbr> ou <a>, pode oferecer definições mais completas e acessíveis.

Atributos

Atributo Valor Descrição
title Texto Fornece uma definição curta ou descrição do termo, exibida como dica (tooltip) quando o cursor passa sobre o termo.
id Identificador único Permite que o termo seja referenciado por outros elementos (por exemplo, <a href="#termo">).

Boas Práticas e Observações

  • Use com significado semântico: Só use <dfn> quando o termo estiver sendo definido, e não apenas mencionado.
  • Termo único: A tag deve envolver apenas o termo em si, não a frase inteira onde ele aparece.
  • Combinação útil: Combine <dfn> com <abbr> para fornecer uma definição completa de siglas e termos técnicos.
  • Acessibilidade: Quando usada corretamente, melhora a compreensão para leitores de tela e indexadores.

Outros exemplos

Aqui estão mais alguns exemplos de uso da tag <dfn>:

Exemplo 1: <dfn> para um termo técnico

<dfn>HTML</dfn> é uma linguagem de marcação usada para criar páginas da web.

Output

HTML é uma linguagem de marcação usada para criar páginas da web

Exemplo 2: <dfn> com atributo title

<dfn title="Cascading Style Sheets">CSS</dfn> é usado para estilizar páginas da web.

Output

CSS é usado para estilizar páginas da web.
