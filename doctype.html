Tag HTML <!DOCTYPE>

Exemplo

Define o tipo de documento e a versão do HTML utilizada.

<!DOCTYPE html>
<html>
   <head>
     <title>Exemplo</title>
   </head>
   <body>
     <p>Olá, mundo!</p>
   </body>
</html>

Definição e uso

A declaração <!DOCTYPE> não é uma tag HTML propriamente dita, mas uma instrução para o navegador sobre qual versão do HTML o documento está usando.

Ela deve aparecer no topo do documento, antes da tag <html>. No HTML5, a declaração é simples e padronizada: <!DOCTYPE html>.

Versões anteriores do DOCTYPE

A declaração DOCTYPE variava conforme a versão do HTML. Veja alguns exemplos históricos:

Versão Declaração
HTML 5 <!DOCTYPE html>
HTML 4.01 Strict <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
HTML 4.01 Transitional <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
XHTML 1.0 Strict <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

Importância

Sem o <!DOCTYPE>, os navegadores podem entrar em quirks mode, renderizando a página com comportamento inconsistente. Por isso, ele garante que o navegador siga o modo de compatibilidade padrão.
