Tag HTML5 <script>

Definição e uso

A tag <script> é usada para **inserir ou referenciar código JavaScript** em páginas HTML5. Pode ser colocada dentro do <head> ou do <body> e suporta tanto scripts internos quanto externos.
Scripts internos são escritos entre <script> e </script>, enquanto scripts externos usam o atributo src.

Estrutura Básica

Sintaxe para script interno:
<script>console.log('Olá Mundo');</script>

Sintaxe para script externo:
<script src="script.js"></script>

Exemplo 1: Script interno

<script>
    alert('Bem-vindo ao site!');
</script>

Exemplo 2: Script externo

<script src="utils/js/app.js"></script>

Exemplo 3: Script no final do <body>

É uma prática comum colocar scripts no final do <body> para melhorar o carregamento da página:

<body>
    <h1>Minha Página</h1>
    <script src="utils/js/app.js"></script>
</body>
